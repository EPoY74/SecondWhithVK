# SecondWhithVK
О проекте

Этот репозиторий содержит Telegram-бота, изначально разработанного для внутренних задач в Тинькофф (ныне Т-Банк).

Основная задача бота — быстрый расчёт процентов от заданной суммы.
Сценарий использования типичный: нужно оперативно посчитать, например:

процент от суммы (комиссия, кэшбэк, доля);

размер начислений, исходя из заданного процента;

другие простые процентные вычисления в рамках рабочих процессов.

Бот взаимодействует с пользователем через Telegram и помогает избегать ручных ошибок при расчётах.
